Import

1. Select main folders (each folder = 1 mode)
2. Structure: Main folder > Color > objects with colors
3. Objects must contain text with numbers (50, 100, 200, etc.)
4. At least 5 colors in each palette
5. Click "Scan Selection"

Import variables from a JSON file, pasted code, or scanned palette.
